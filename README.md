# 🌍 NGO Website

This project is a responsive NGO (Non-Governmental Organization) website designed to create awareness, showcase activities, and encourage donations and volunteer participation.

## 🚀 Features

- 🏠 Home page with NGO mission & vision
- 📖 About section describing the organization
- 🎯 Services / Activities section
- 💰 Donation section
- 🤝 Volunteer registration
- 📞 Contact form
- 📱 Fully responsive design (mobile + desktop)

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap (if used)
